# 🦍 Donkey Kong (동키콩 모작)

## 🎮 개요
 - 게임 장르: 플랫포머 게임
 - 플랫폼: PC
 - 플레이 인원: 1인
 - 1인 개발
 - **구현 내용**:
   > + **WinAPI를 활용**하여 2D 게임 개발
   > + 스프라이트 애니메이션, 인트로, 엔딩씬 애니메이션 (→ **더블 버퍼링** 적용)
   > + 총 5개의 스테이지(+ 스테이지 별 새로운 장애물들), 망치 아이템
   > + 사운드(+ 소리 조절 기능-배경음/효과음 개별적으로), 일시정지 기능, 다시시작 기능
 - 개발에 사용된 기술 스택:
   + WinAPI
   + C++
 - 시연 영상: [▶️ 유투브 영상 보기](https://www.youtube.com/)
