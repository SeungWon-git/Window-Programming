* 사용 버튼: 
    W A S D - 이동키
    ENTER - 총알 발사
    SHIFT - 특수 총알 발사 (아이템 사용)
    I - 무적상태 ON/OFF
    P - 칸나누기 보기 ON/OFF
    Q - 나가기

* 미 구현 내용:
    1. 난이도 조절
    2. 이펙트 (캐릭터, 적 사망, 총알 이동 그라데이션)
    * 나머지는 모두 구현하였습니다.

* 게임 설명:
    [필드] - 맨 왼쪽, 오른쪽, 위, 아래 는 총알로 파괴되지 않으며 이동할 수도 없는 벽입니다.
    [적 생성] - 적은 벽(파괴 가능한 벽 + (위에서 설명한)파괴 불가능한 벽)에서만 생성되며, 여러 벽들 중에 랜덤한 위치에서 생성됩니다.
        (단 플레이어가 너무 인접한 위치에서는 생성X)
    [폭발] - 적이 사망시 폭발하며 폭발 범위내 모든 적들도 폭발하게되어 연쇄 폭발을 일으킵니다.
    [사망] - 주인공의 목숨은 총 2개이며 하단에 표시됩니다. 적과 만나게되면 목숨이 하나 소모되며, 게임이 리셋이되고 캐릭터는 랜덤한 위치에서 다시 시작합니다.
    [콤보] - 밑에 콤보 게이지가 다 소모되기 전에 적을 피격시 콤보가 쌓이며 콤보가 쌓일 수록 획득 점수가 높아지고 콤보 게이지가 더 빨리 소모된다.
    [아이템] - 아이템을 사용하면 특수 총알을 발사합니다. 아이템은 총 3개까지 획득 가능하며 한번 사용시 현재 가지고있는 아이템을 모두 사용합니다.
        아이템은 적을 처치시 랜덤한 확률로 1~3개 까지 생성되며 적이 사망한 자리에서 생성됩니다. 특수총알은 일반 총알과 같은 역활이지만 많이 
        모아 사용할 수록 넓게 퍼져서 날라갑니다.  
    [시간이 지날 수록] - 시간이 지날 수록 적의 생성 간격이 짧아지며 적의 스피드도 빨라집니다.
